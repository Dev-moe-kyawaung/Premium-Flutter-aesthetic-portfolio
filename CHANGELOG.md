## [2.0.0] — 2026-06-22

### Added
- **V2 flagship design** (`index-v2.html`) — "Android flagship launch" concept
- Live Kotlin typewriter terminal in hero section
- Bento-grid tech stack layout with 12-column asymmetric cells
- Mixed-size project grid (featured 8-col + 4-col cards)
- Animated skill progress bars in bento cells
- Marquee tech strip between hero and stack sections
- Active nav link highlighting on scroll
- `Space Grotesk` + `DM Sans` + `Fira Code` type system
- Material Dynamic Color teal `#00E5FF` palette
- Scanline overlay texture for premium display-screen feel
- Animated background orbs with CSS keyframes
- Split sticky experience layout (left pins, right scrolls)
- PWA manifest (`manifest.json`)
- GitHub Actions CI/CD auto-deploy workflow
- Custom 404 page with Kotlin exception syntax
- MIT License
- This CHANGELOG

### Changed
- Hero redesigned from static widget tree to animated typewriter terminal
- Nav redesigned as floating pill (Linear/Vercel style)
- Projects section uses mixed-size grid instead of uniform cards
- Experience section uses two-column sticky layout
