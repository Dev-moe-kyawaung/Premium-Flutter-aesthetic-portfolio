## 🌐 Portfolio Website

> This repository hosts my **ultra-premium personal portfolio** — a Flutter/Material You-inspired single-file HTML site, deployable directly to GitHub Pages with zero build steps.

### ✨ Features
- 🎨 Flutter-inspired design system with Material You color tokens
- ⌨️ Live Kotlin typewriter terminal in the hero section
- 📱 Fully responsive — mobile, tablet, desktop
- 🚀 GitHub Actions CI/CD — auto-deploys on every push to `main`
- ⚡ Zero dependencies — single `index.html`, no npm, no build tools
- 🌙 Dark mode native — no light mode toggle needed
- ♿ Accessible — keyboard navigable, reduced motion respected
- 🔧 PWA-ready manifest included

### 🗂️ Repository Structure

```
moe-kyaw-aung-portfolio/
├── index.html              # ← Main portfolio (deploy this)
├── index-v2.html           # ← V2 variant (flagship Android launch concept)
├── 404.html                # ← Custom GitHub Pages 404
├── manifest.json           # ← PWA manifest
├── .github/
│   └── workflows/
│       └── deploy.yml      # ← Auto-deploy to GitHub Pages
├── README.md               # ← This file
├── CHANGELOG.md            # ← Version history
├── LICENSE                 # ← MIT License
└── .gitignore              # ← Git ignore rules
```

### 🚀 Deploy to GitHub Pages

**Option A — Automatic (recommended):**
1. Fork or push this repo to your GitHub account
2. Go to **Settings → Pages**
3. Set source to **GitHub Actions**
4. Push any change to `main` — the workflow auto-deploys ✅

**Option B — Manual:**
1. Go to **Settings → Pages**
2. Set source to **Deploy from a branch**
3. Select `main` branch, `/ (root)` folder
4. Your site will be live at `https://dev-moe-kyawaung.github.io/moe-kyaw-aung-portfolio/`
